## Features
- Responsive design
- Contact form with validation & fake submit (ready for EmailJS)
- Modern UI with TailwindCSS

## Run Locally

**Prerequisites:** Node.js (v18+)

1. Clone the repo
2. Install dependencies:
   ```
   npm install
   ```
3. Copy `.env.example` to `.env.local` and add your API key if needed
4. Run dev server:
   ```
   npm run dev
   ```
5. Open http://localhost:5173

## Build & Deploy
```
npm run build
npm run preview
```

Deploy to Vercel/Netlify/Vite Preview easily.
